## 1. 项目概述

### 1.1 项目简介

DataRelationshipDesign 是一款**工业级数据库可视化设计工具**，提供直观的拖拽式界面，支持数据库表结构设计、关系管理、DDL代码生成等核心功能。

### 1.2 核心价值

| 维度 | 说明 |
|------|------|
| **可视化设计** | 拖拽式操作，所见即所得 |
| **多数据库支持** | MySQL / PostgreSQL / SQLite / SQL Server / Oracle |
| **团队协作** | 实时同步 + 权限管理 + CRDT |
| **工程化** | 版本控制 / 分支管理 / Git集成 |
| **AI赋能** | 智能表结构生成与数据模拟 |

### 1.3 目标用户

- 数据库开发工程师
- 后端开发工程师
- 系统架构师
- 技术团队负责人
- DBA（数据库管理员）

## 3. 功能模块

### 3.1 可视化画布 (Canvas)

- 无限画布，支持缩放和平移
- 表节点拖拽布局
- 多选、批量操作
- 网格对齐与辅助线
- 撤销/重做 (Ctrl+Z / Ctrl+Shift+Z)
- 快捷键支持（新建表 Ctrl+T、保存 Ctrl+S 等）

### 3.2 表设计 (Table Designer)

- 创建、编辑、删除表
- 字段增删改查
- 主键(PK) / 外键(FK) / 唯一约束设置
- 自增字段、默认值、非空约束
- 字段注释与排序
- 索引设计（BTREE/HASH/FULLTEXT）

### 3.3 关系设计 (Relationship)

- 可视化外键关系连线
- 三种关系类型：一对一(1:1)、一对多(1:N)、多对多(M:N)
- 级联规则配置（CASCADE / SET NULL / RESTRICT / NO ACTION）
- 循环依赖检测
- 关系线智能避让算法

### 3.4 DDL 生成器

支持 5 种主流数据库：

| 数据库 | 支持状态 | 特性 |
|--------|----------|------|
| MySQL | ✅ 完整 | AUTO_INCREMENT、COMMENT、ENGINE |
| PostgreSQL | ✅ 完整 | SERIAL、JSONB、UUID |
| SQLite | ✅ 完整 | AUTOINCREMENT、WITHOUT ROWID |
| SQL Server | ✅ 完整 | IDENTITY、NVARCHAR |
| Oracle | ✅ 完整 | SEQUENCE、NUMBER、VARCHAR2 |

**额外能力**：
- 增量 DDL 生成（ALTER TABLE）
- DDL 格式化与语法验证
- 表前缀 / Schema 配置
- 大小写关键字选项

### 3.5 E-R 图导出

采用 **Chen Notation** 标准：

- **实体(Entity)**: 蓝色矩形，显示表名和注释
- **关系(Relationship)**: 橙色菱形，显示基数类型
- **属性(Attribute)**: 字段列表，含类型和约束
- **主键标识**: 下划线 + 加粗 + 🔑 图标
- **外键标识**: 斜体 + 🔗 图标
- **基数标注**: 连线端点圆形标签 (1/N/M)
- **导出格式**: PNG (2x高清) / SVG (矢量)
- **安全防护**: XSS 转义 (escapeXml)

### 3.6 版本管理

- 设计版本快照与回滚
- 分支管理（类似 Apifox 分支模式）
- 可视化版本对比（表结构差异高亮）
- Git 配置集成（自动提交/推送）

### 3.7 团队协作

- 用户注册/登录 (JWT + bcrypt)
- 团队创建与管理
- 项目成员角色（Owner / Editor / Viewer）
- 邀请码机制
- **实时协作** (WebSocket + Yjs CRDT):
  - 操作实时同步
  - 协作光标显示
  - 字段级锁机制
  - 离线编辑 + 自动合并

### 3.8 数据库逆向工程

- 连接现有数据库（MySQL / PostgreSQL / SQLite）
- 导入表结构、字段、主键、外键、索引
- 类型智能映射
- 本地 SQLite 文件直接导入
- 连接配置管理与测试

### 3.9 AI 智能助手

- LLM 辅助生成表结构
- Ollama 本地大模型支持
- 批量模拟数据生成
- RAG 向量记忆（可选）

### 3.10 其他功能

| 功能 | 描述 |
|------|------|
| 主题系统 | 浅色 / 深色 / Darcula / 蓝色 (4种) |
| 导入导出 | JSON / SQL 格式 |
| 评论标注 | 表/关系级别评论讨论 |
| 操作历史 | JSON/CSV 导出 |
| 离线模式 | IndexedDB 本地存储 |
